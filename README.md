### 项目简介
```
目标：
    模拟商城，完整的购物流程、后端运营平台对前端业务的支撑，和对项目的运维，有各项的监控指标和运维指标。
技术点：
    核心框架：Spring Cloud、Spring Boot
    安全框架：Spring Security Oauth2
    网关框架：Spring Cloud Gateway
    持久层框架：MyBatis Plus、PageHelper
    数据库连接池：Alibaba Druid
    中间件：Redis、RocketMQ
    日志管理：Logback
    小程序框架：uniapp
```
